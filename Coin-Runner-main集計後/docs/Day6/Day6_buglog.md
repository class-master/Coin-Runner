# Day6 バグログ（Coin-Runner）

このファイルは Day6 の 3時間で見つかったバグや気づきをメモするためのものです。

## 記入ルール

- 1つのバグにつき 1 見出し
- 「再現手順」「原因」「対応」「メモ」の4つを書く
- 修正が終わったものは先頭に `[fixed]` を付ける

---

## 例：

### [fixed] best_score.json が壊れていると起動で落ちる

- 再現手順  
  1. `save/best_score.json` を開く  
  2. 中身を `{` だけにして保存する  
  3. アプリを起動する  
- 原因  
  - `json.load()` の例外を握っていなかった  
- 対応  
  - `load_best()` 内で try/except し、失敗時は best=0 にして続行  
- メモ  
  - 「落ちない設計」はゲーム制作で超大事。まず続行できること。
