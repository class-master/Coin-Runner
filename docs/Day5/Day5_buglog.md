# Day5 バグログ（Coin-Runner）

このファイルは Day5 の 3時間で見つかったバグや気づきをメモするためのものです。

## 記入ルール

- 1つのバグにつき 1 見出し
- 「再現手順」「原因」「対応」「メモ」の4つを書く
- 修正が終わったものは先頭に `[fixed]` を付ける

---

## 例：

### [fixed] GameOver 後に Score がリセットされない

- 再現手順  
  1. コインを3つ取ってから障害物に当たって GameOver にする  
  2. Retry してもう一度プレイを始める  
- 原因  
  - `ScoringService.reset()` が Retry 時に呼ばれていなかった  
- 対応  
  - `Play` 画面のリスタート処理の中で `scoring.reset()` を呼ぶようにした  
- メモ  
  - 「現在スコア」と「ベストスコア」でリセットしたいものが違うことに注意する。
